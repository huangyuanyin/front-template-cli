# front-template-cli

一个基于 node 的 cli 工具，用于从远程仓库中拉取项目模板，并以此模板，快速创建项目

## 使用方式

### 安装

**npm**

```shell
# 安装
npm install -g front-template-cli
# 卸载
npm uninstall front-template-cli -g
```

**pnpm**

```shell
# 安装
pnpm add front-template-cli -g
# 卸载
pnpm remove front-template-cli --global
```

### 使用

**终端**
```
ftc
```

## 本地开发

```shell
# 安装
pnpm install
# 打包
pnpm run build
```
